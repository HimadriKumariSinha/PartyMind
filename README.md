# PartyMind – Party Theme Generator

## Overview

PartyMind is a web-based application designed to generate creative and personalized party themes based on user inputs such as occasion, age group, and preferences. The system processes these inputs and provides relevant theme suggestions along with supporting recommendations.

## Features

* Dynamic generation of party themes based on user input
* Suggestions for decorations, color schemes, and overall setup
* Recommendations for music and activities
* Multi-step interactive user interface
* Light and dark mode support

## Tech Stack

* HTML
* JavaScript (React via CDN)
* JSX
* Tailwind CSS

## Project Structure

* `PartyThemeGenerator.html` – Main user interface
* `party_theme_generator_premium.jsx` – Core application logic

## How to Run

1. Clone or download the repository
2. Open `PartyThemeGenerator.html` in any modern web browser
3. Use the interface to input details and generate themes

## System Approach

The application uses a rule-based decision system to process user inputs and generate appropriate party themes. Based on predefined logic and structured conditions, the system produces context-aware recommendations. This approach demonstrates fundamental concepts of intelligent systems, including input-driven decision-making and automated suggestion generation.

## Example

**Input:**

* Occasion: Birthday
* Age Group: Teens

**Output:**

* Theme: Neon Glow Party
* Suggestions: LED lighting, electronic music, vibrant dress code

## Future Scope

The system can be extended by integrating advanced AI techniques such as natural language processing and external APIs to generate more dynamic and context-aware themes.
